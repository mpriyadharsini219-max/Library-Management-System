# Library Management System (C)

##  Description
A simple file-based Library Management System developed in C to manage book records using file handling.

##  Features
- Add book details
- View all books
- Search book by ID
- Delete book record

## Technologies Used
- C Programming
- File Handling
- Structures

##  File Storage
- Data is stored in `library.txt`

##  How to Run

1. Compile:
   gcc main.c -o library

2. Run:
   ./library
